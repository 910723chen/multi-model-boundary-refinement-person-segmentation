# Dataset and Privacy Notice

This repository includes a small selected test set under:

```text
test_images/final_selected/
```

These images are included to make the course-project results inspectable. They should be treated as course demonstration data, not as an open public dataset.

If this repository is made public, confirm that the people shown in the photos have agreed to this use. If public sharing is not appropriate, remove `test_images/final_selected/` before publishing and keep only the derived summary results under `results/`.

The reported experiment uses the 12 included images only as a small preliminary test set. The project does not claim general benchmark performance.
